# Weather

The Weather module shows current conditions and a short forecast on the panel,
using the free [Open-Meteo](https://open-meteo.com) service.

## Set your location

In **Settings → Weather** you can either:

- **Enter a city or coordinates** manually, or
- **Use automatic location**, which estimates your city from your device's
  public IP address via [ipwho.is](https://ipwho.is).

Only the location you choose is sent to Open-Meteo to retrieve the forecast. No
account or API key is required.

## What it shows

- Current temperature and conditions.
- A short hourly / daily outlook.

See the [Privacy Policy](../privacy/) for the exact data each request sends.
