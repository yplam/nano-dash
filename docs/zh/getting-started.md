# 快速开始

NanoDash 将一块小巧的圆形触摸屏变成你桌面上的第二块「速览」屏幕——天气时钟、
计时器、电脑资源监视器、媒体控制等等——由一个跨平台桌面应用驱动。

## 安装应用

在[发布页面](https://github.com/yplam/nano-dash/releases)下载适合你系统的最新版本：

- **Windows**——`.msix` 安装包。
- **macOS**——`.dmg` 安装包。
- **Linux**——Flatpak 安装包。

你可以单独运行 NanoDash，在电脑屏幕上预览仪表盘；外接面板是可选的。

## 连接面板

1. 将 NanoDash 触摸屏插入 USB 接口。
2. 启动应用——它会自动识别面板，并开始把仪表盘镜像到面板上。
3. 在面板上左右滑动可切换页面；页面内的内容可上下滚动。

> 如果面板未被识别，请确认没有其他程序正在占用该 USB 设备，然后重新插拔。

## 配置你的仪表盘

打开**设置**，选择要显示哪些模块及其顺序，并填写各模块所需的信息（天气所需的
位置、日历地址、行情自选列表等）。你填写的所有内容都保存在本地电脑上——具体每个
模块的行为请参见[隐私政策](../privacy/)。
