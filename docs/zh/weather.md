# 天气

天气模块在面板上显示当前天气状况和短期预报，数据来自免费的
[Open-Meteo](https://open-meteo.com) 服务。

## 设置位置

在**设置 → 天气**中，你可以：

- **手动输入**城市或经纬度，或
- **使用自动定位**，通过 [ipwho.is](https://ipwho.is) 依据你设备的公网 IP
  地址估算所在城市。

只有你所选择的位置会被发送给 Open-Meteo 以获取预报。无需账号或 API 密钥。

## 显示内容

- 当前温度与天气状况。
- 简短的逐时／逐日预报。

各请求发送的具体数据请参见[隐私政策](../privacy/)。
