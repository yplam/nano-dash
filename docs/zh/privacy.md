# NanoDash 隐私政策

**最后更新：2026 年 7 月 14 日**

NanoDash（下称「本应用」「我们」）是一款桌面应用，用于显示由若干小部件组成的
仪表盘——时钟、日历、天气、行情、系统监视器、媒体「正在播放」、计时器／秒表、
Live2D 角色，以及可选的本地语音助手——并可将该仪表盘镜像到外接的 USB 触摸屏。

我们高度重视你的隐私。本政策说明本应用会处理哪些数据、这些数据流向何处，以及你
拥有哪些选择。

## 概要

- **我们没有服务器，也不收集、存储或出售你的个人数据。** NanoDash 没有用户账号，
  无需注册。
- **你的所有设置都保留在你的设备上。** 配置（仪表盘布局、API 密钥、日历地址、
  自选列表等）都保存在你电脑的本地存储中。
- **本应用不含任何广告，也没有分析或跟踪 SDK。**
- 某些可选功能会在你主动启用后，直接从你的设备连接到你所选择的第三方服务以获取
  信息。这些连接在下文中说明。

## 保存在你设备上的数据

以下内容保存在你电脑的本地（通过操作系统标准的应用存储），绝不会传输给我们：

- 你的仪表盘配置和已启用的模块。
- 功能设置，包括你填写的任何 API 密钥、接口地址、日历订阅地址、位置／城市、
  行情自选代码，以及语音／智能体偏好。

你可以随时通过清除应用设置或卸载应用来删除这些数据。

## 应用在本地处理的信息（绝不离开你的设备）

某些模块会读取你电脑上的信息仅用于显示。这些数据在设备本地处理，**不会**发送给
我们或任何第三方：

- **系统监视器**——你电脑的 CPU、内存和网络使用情况。
- **正在播放**——你电脑上当前播放媒体的元数据（如曲目标题／艺术家），读取自
  操作系统的媒体会话。
- **用量监视器**——你电脑上安装的开发者命令行工具（如 Claude Code、Codex）本地
  存储的用量／限额日志。
- **语音助手**——启用后，麦克风音频会在**你的设备上**采集并处理（语音活动检测、
  语音识别和语音合成均在本地运行）。音频不会上传给我们。（若你将其连接到外部
  语言模型，请参见下文第三方服务中的语音助手说明。）

## 第三方服务（可选功能）

当你启用某些模块时，本应用会**直接从你的设备**连接到以下第三方服务以获取信息。
我们不运营这些服务，也不会从它们那里收到任何数据。每项服务均受其各自提供商的
隐私政策约束。

| 功能 | 连接的服务 | 发送的数据 | 用途 |
|---|---|---|---|
| 天气 | Open‑Meteo（`open-meteo.com`） | 你所选位置的坐标／城市 | 获取天气预报 |
| 天气定位 | ipwho.is（`ipwho.is`） | 你设备的公网 IP 地址 | 在使用自动定位时估算你的城市 |
| 行情 | Yahoo Finance（`finance.yahoo.com`） | 你自选列表中的股票／加密货币／外汇代码 | 获取报价 |
| 日历 | **你所配置的**日历服务器（任意 iCalendar/ICS 或 CalDAV 地址） | 订阅地址及你填写的任何凭据 | 获取你的日历事件 |
| 语音助手／AI 智能体 | **你所配置的** AI 提供商（OpenAI 兼容接口，如 `api.openai.com`，或 Anthropic `api.anthropic.com`） | 你的提示词／对话文本及你的 API 密钥 | 生成助手回复 |

说明：

- **这些连接由你掌控。** 除非你启用相应模块并（在适用时）提供接口／凭据，否则
  不会连接任何第三方服务。
- **你填写的 API 密钥和日历凭据**保存在本地，且仅发送到你所配置的接口，用于对你
  自己的请求进行身份验证。
- **AI 智能体／语音提供商：** 如果你将助手连接到外部语言模型，你的对话文本（若
  使用语音，则为转写后的文本）会发送给你所选择的提供商。请查阅该提供商的隐私
  政策。如果你未配置外部提供商，则不会有任何对话数据离开你的设备。

## 应用可能请求的权限

- **麦克风**——仅在你启用语音助手时，用于本地语音处理。
- **网络访问**——用于连接上文所列的可选第三方服务。
- **USB 设备访问**——用于与已连接的 NanoDash 触摸屏硬件（若存在）通信。除了你
  已在屏幕上看到的仪表盘画面外，不会向该设备发送任何个人数据。

## 儿童隐私

NanoDash 并非面向儿童设计，也不会在知情的情况下收集任何儿童的个人信息。
